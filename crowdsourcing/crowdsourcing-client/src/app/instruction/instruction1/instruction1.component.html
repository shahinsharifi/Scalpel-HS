Overview

Thank you for your participation!


Given a natural scene image (E.g. living room), the goal of this task is two fold:

1. validate the correctness of relationships between objects that are captured in the scene.


2. select (or add if necessary) a minimal set of relationships that are sufficient to identify the given image label


Now, please click on the 'Next' button and read the following instructions step-by-step.
Check A Relation
Step 1: Check a relation

During this step, you are required to validate all the relations in the Relation List (on the right side of the page).

First, move your mouse on the relation that needs your validation, then you will see two bounding boxes appear on the Original Image.
Please note that the color of each bounding boxes is the same as the corresponding object label in the Relation List.

Then, you should validate whether the description of the relation correctly corresponds to the bounding boxes or not.
Validate A Relation
Step 2: Validate a relation

Now, you need to validate whether the relation is correct or wrong (Don’t worry if you click a button that you do not aspire by mistake, you can edit your answers later).

If you think it is correct, please suggest whether the the relation is typical. If the relation is unusual, please give your reason.


Then, you will be asked to determine how relevant this relation is in identifying the given label (e.g. child room) of the scene .

 Validate All the Relations
Step 3: Validate all the relations

Repeat Step 2 until there are no pending relations that need validation (and the following window will pop up). If you think that the relations you have validated are not sufficient for describing the given scene image, please add new relations that you think are missing.

If you think that the given label is not correct, please give your reason (and the correct label) and submit the task.
Add New Relation
Step 4: Add new relation

In order to add a new relation to correctly describe the given scene image, first, you are asked to draw a bounding box of an object in the relation you want to add.

Then you should select the name of this object from a given list. If you can’t find the object name you want, please select ‘object’.

After that, you are asked to determine whether this object is typical, and what color it is (optional). Next, select the type of the relation from a given list.

Similarly, please define the second object in this relation, by repeating the previous procedure (draw a bounding box, select an object name, determine whether it is typical, select the right color). Then, rate the importance of this relation, give your reason, and add this relation. Please add a relation which is relevant to identify the scene. E.g. "sofa" for identifying living room
Define Minimal Set
Step 5: Define a minimal identifying set

After you validate all the relations, please select the relations to form a minimal set that can sufficiently identify the scene. Also, during this step, you can edit your answers.
Submission
Step 6: Submit your answers

This is the final step before submission. During this step, please choose all objects which are specifically relevant to the given scene.
After that, you can submit this task.

Bonus!

You will receive additional bonuses based on the quality with which you complete the tasks. The bonus payment will be based on the following factors:

1. The validity of the new relations added in the image.


2. The accuracy of bounding boxes corresponding to the new relations added.


3. The quality of the reasons provided.
