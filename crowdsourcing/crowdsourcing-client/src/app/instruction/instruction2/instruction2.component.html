Overview

Thank you for your participation!


The goal of this task is to identify objects and relations in a scene shown in an image that an automatic model has focused on (hot zones). This is displayed using a ‘heat map’ over the image. The areas in the image which correspond to darker shades of red color indicate the objects and relations that the model has focused on. We are interested in also understanding which of these objects or relations in the hot zones are not important to identify the scene in the image.




Now, please read the following instructions to learn how to finish this task.
Identify the Objects
Step 1: Identify the Objects

During this step, you are asked to draw bounding boxes to indicate all the objects/relations that comprise such hot zones.

To do this, use your mouse to draw a bounding box on the heatmap/original image. You can switch between "Heatmap" and "Original Image" modes by clicking on the tab label above the image viewer.

Then you should select the name of this object from a given list. If you can’t find the object name you want, please select ‘object’.

Then you should select the name of this object from a given list. If you can’t find the object name you want, please select ‘object’.

Then you should select the name of this object from a given list. If you can’t find the object name you want, please select ‘object’.
Identify the Relations
Step 2: Identify the Relations

During this step, you are asked to identify relations existing in the hot zones, after you have identified at least two objects. Please add as many relations as are meaningfully present between the objects in the hot zones.
Judge the Relations
Step 3: Judge the Relations

Please judge whether each object in the hot zone goes against your idea of what you would expect to find in, for example, "living room" by clicking the Judge button.

In this example, ‘painting on wall’ is not an essential relation for identifying dining rooms, since the wall of a bedroom or a living can also have paintings. Please provide a brief explanation for the score assigned.
Submission
Step 6: Submit your answers

This is the final step before submission. During this step, please suggest whether the scene shown in this image is typical. If you think that the image is an unusual example of the scene, please explain why.
After that, you can submit this task.

Bonus!

You will receive additional bonuses based on the quality with which you complete the tasks. The bonus payment will be based on the following factors:

1. The validity of the new relations added in the image.


2. The accuracy of bounding boxes corresponding to the new relations added.


3. The quality of the reasons provided.
