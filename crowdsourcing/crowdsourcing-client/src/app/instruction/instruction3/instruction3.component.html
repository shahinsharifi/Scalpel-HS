A high-quality image dataset can radically improve the performance of computer vision models. Many errors made by machine learning models are a result of under-represented types of images in the dataset. Such under-represented types of images are called unusual or atypical, with respect to the images in the dataset.

To help us improve the quality of the machine learning models that automatically detect objects in images, in this task we ask you to:

  • First, determine whether the given image is about the given class or contains the object(s) of the given class;
  • Then, determine the degree of typicality of the image with respect to the dataset using a 7-point Likert-scale (from Highly Typical to Highly Atypical). Here you need to judge whether the given image (shown on the left-hand side of your screen in the task) is typical with respect to the dataset.

In the task, to help you make better decisions, we also show some images (with their classes shown in the “Visually Similar” tab on the right-hand side of your screen in the task). These images look visually the most similar to the given image, and the machine learning model is likely to consider this image to belong to one of these classes.

Furthermore, to help you form an understanding of the given image class, we show some sample images of the given class from the dataset (but not necessarily visually similar).

A screenshot of the task.
  • A typical image means there are other similar images present in the sample set shown above, and many visually similar images are of the same class;
  • An atypical image means the image is unusual with respect to the images present in the sample set, and many visually similar images belong to different classes.
